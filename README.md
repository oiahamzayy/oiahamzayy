## About Me

I'm a **Senior PHP & WordPress Developer** with deep expertise in building production-grade plugins, WooCommerce extensions, headless WordPress architectures, and scalable REST APIs. I write clean, testable, standards-compliant code that ships on time and survives long-term maintenance.

- **WooCommerce** — custom extensions, payment gateways, subscriptions, dynamic pricing
- **Plugin Development** — OOP PHP, Composer, PSR standards, WordPress Coding Standards
- **REST API & Headless** — custom endpoints, JWT auth, decoupled frontends with Next.js
- **Custom Themes** — full-site editing, Gutenberg block development, headless themes
- **Database** — complex `$wpdb` queries, custom tables, query optimization
